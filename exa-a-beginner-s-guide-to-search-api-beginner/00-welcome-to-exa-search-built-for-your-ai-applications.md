# Welcome to Exa: Search Built for Your AI Applications

### What Exa Is and Why It Exists

Exa is a search engine built specifically for artificial intelligence rather than for humans clicking through web pages. The problem it solves is simple: traditional search gives you a list of links meant for people to browse. If you are building software, an automated assistant, or any application that needs live knowledge from the internet, you need information in a clean, structured form that your program can actually use. Exa acts as a specialized research layer that fetches exactly what your application needs from the web.

### Who This Is For and What You Can Build

This is for developers, researchers, and product builders who want their applications to access the web without writing fragile scraping scripts or parsing cluttered pages by hand. Exa fits wherever an application needs to answer questions using current web knowledge. It enables automated research, factual lookups, content enrichment, and smart agents that can summarize what they find. Whether you need to find specific writing on a topic, discover pages that share the same meaning as one you already trust, or pull readable text from complex websites and PDFs automatically, Exa makes those applications possible without manual busywork.

### Why Choose Exa Over a Generic Search or DIY Workaround

A generic web search returns results optimized for human browsing—ads, SEO pages, and formatting meant for eyes, not for software. Building your own workaround means handling broken layouts, pages rendered by complex scripts, and websites that change without warning. Exa handles that complexity for you. You ask a question in plain language, and Exa returns focused, relevant results. It can retrieve clean content from pages automatically, pick out the key excerpts that answer your query while using far less data, and find conceptually similar pages when you need more of the same.

This course begins with the big picture of how AI-friendly search works, moves into the core concepts behind finding and retrieving web content, and then shifts to practical usage so you can confidently build it into your own projects.
